Browser graphics experiment

ASCII Point Cloud Studio

Turn colored point data into animated 3D ASCII art. Load any compatible JSON file, preview it with the reusable web component, or edit and export the points in the browser.

Open the editor View source

Included examples

One renderer, different point sets

Use the component

Import one module and point the element at any compatible JSON file.

<script type="module"
  src="./src/ascii-point-cloud.js">
</script>

<ascii-point-cloud
  src="./examples/points.json"
  speed="1"
  depth="0.05">
</ascii-point-cloud>

Point format

Each entry contains a 2D position followed by RGB color channels. Bounds are calculated automatically.

[
  [x, y, r, g, b],
  [x, y, r, g, b]
]
